# Fizz Buzz

## Development

Install npm packages `npm install`

Build with `npm run build`

Run with `node dist/index.js -f=1 -t=100`

`-f` is the number to process from
`-t` is the number to process to

## Installing

Build with `npm run build`

`npm install -g .`

Run with:
`getFizzbuzz -from=1 -to=100`
